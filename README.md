# Избранные квартиры Циан — карта (Санкт-Петербург)

Карта избранных объявлений с Циан на OpenStreetMap: сайдбар с превью, рейтингом, сортировкой; по клику на превью — полноэкранный слайдер фото и описание.

## Быстрый старт

1. **Парсинг избранного** (страница `data/favorite.htm`):
   ```bash
   python parse_cian_favorites.py
   ```
   Извлекаются: ссылка, превью, заголовок, **полная цена**, **полный адрес** (как на сайте), **метро** (станция + время), телефон, описание. Результат: `apartments.json`. Нужен `pip install beautifulsoup4`.

2. **Геокодирование** (координаты для OpenStreetMap):
   ```bash
   python geocode_cian.py
   ```
   Используется Nominatim (бесплатно). Для более точных координат по СПб задайте переменную окружения `YANDEX_GEO_API_KEY` и снова запустите скрипт. Нужен `pip install requests`.

3. **Генерация карты**:
   ```bash
   python create_map_cian.py
   ```
   Создаётся `map_cian.html` с картой, сайдбаром (адрес, метро, рейтинг, «Показать все фото») и слайдером по клику на превью.

3. **Открыть карту**: двойной клик по `map_cian.html` или:
   ```bash
   start map_cian.html
   ```

## Полные фото и описание по каждой квартире

- **Автоматически** (может не сработать из‑за защиты Циан):
  ```bash
  python fetch_cian_offers.py
  ```
  Скрипт качает страницы объявлений по ссылкам из `apartments.json` и пытается вытащить все фото. Затем снова запустите `create_map_cian.py`.

- **Вручную** (надёжно): для каждой из 15 квартир откройте ссылку на Циан → «Файл» → «Сохранить как» → «Веб-страница, полностью». Сохраните файлы в папку `data/offers/` с именами вида `offer_324037941.html` (число — ID из URL объявления). Затем:
  ```bash
  python parse_cian_offer_pages.py
  python create_map_cian.py
  ```
  В карте по клику на превью откроется слайдер со всеми фото и полным описанием.

## Структура

- `data/favorite.htm` — сохранённая страница «Избранное» Циан (со статикой в `data/favorite_files/`).
- `data/offers/` — сюда класть сохранённые HTML страницы объявлений (по одной на квартиру).
- `apartments.json` — данные квартир (ссылка, превью, цена, адрес, координаты, фото, описание).
- `map_cian.html` — готовая карта (Leaflet + сайдбар + модальное окно-слайдер).

Картинки в сайдбаре берутся из `data/favorite_files/` (пути в JSON заданы относительно корня папки `cian`). Если открывать `map_cian.html` по протоколу `file://`, браузер может блокировать загрузку — тогда поднимите простой HTTP-сервер в папке `cian`, например: `python -m http.server 8000` и откройте `http://localhost:8000/map_cian.html`.

## Публикация на GitHub Pages

Чтобы смотреть карту по ссылке и делиться ею:

1. **Обязательно добавьте в репозиторий файл данных** (иначе на сайте будет пустой сайдбар):
   ```bash
   git add data/apartments.js
   ```
   В `.gitignore` папка `data/` игнорируется, кроме `data/apartments.js` — этот файл нужен для загрузки списка квартир на GitHub Pages.

2. Закоммитьте и запушьте репозиторий (включая `map_cian.html`, `index.html`, папки `css/`, `js/`).

3. В репозитории: **Settings** → **Pages** → **Source**: **Deploy from a branch** → **Branch**: `main`, папка **/ (root)** → **Save**.

4. Через 1–2 минуты сайт будет доступен:
   - `https://<ваш-логин>.github.io/<имя-репозитория>/` или `.../map_cian.html`

Папки с фото (`data/*_files/`, `data/static_2/`) в репозиторий не попадают — на публичной странице превью квартир не отобразятся, но карта, список, ссылки на Циан и рейтинг будут работать. Чтобы и фото были на сайте, уберите из `.gitignore` правило для `data/` и добавьте нужные папки с картинками в git (репозиторий станет больше).
